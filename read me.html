Логика URI такая: на первом месте идёт /admin/ или /fr/

Если язык передан НЕправильно, то он становится контроллером.

Задача котроллера - сформировать данные, которые потом будут переданны представлению (view).

С БД работают только модели, они записывают в БД и получают из БД данные. Контроллеры обращаются к ним при необходимости.



Урок 2: Создание структуры каталогов

config		- хранит настройки.
controllers	- хранит классы контроллеров.
models		- хранит классы моделей.
views		- хранит виды или представления, то есть шаблоны, в папке views для каждого контроллера создана одноименная папка.
lib		- библиотеки фреймворка, вспомогательные коды, коды основных классов приложения.
lang		- мультиязыковые настройки.
webroot		- содержимое с публичным доступом.
	- css
	- js
	- img
	- uploads


Урок 3: Обработка и перенаправление запросов

Делаем единую точку входа - файл index.php, который хранится в папке /webroot/.

Сначала перенаправляем запросы на этот файл при помощи конфигурационного файла .htaccess сервера Apache.

Первый файл .htaccess, который в корневой директории, перенаправляет все запросы в папку /webroot/.

Второй файл .htaccess решает к чему направлялся запрос. Если запрашиваемый файл существует, идёт работа с ним, иначе запрос передаётся на index.php.



Урок 4: Добавление ключевых классов. Автозагрузка классов.

В папке /lib/ создали класс config.class.php. Методы set() и get() - это всё, что нужно для работы с этим классом. Файл /config/config.php через set() устанавливает их. Почему делается так, а не напрямую в классе, как свойства, я не понял.

В папке /lib/ создаём файл router. Парсинг не делаем.

В папке /lib/ создаём файл init. В нём создаём функцию __autoload().

Классы могут быть только в трёх папках. Имена классов:

  • /lib/***.class.php
  • /controllers/***.controller.php
  • /models/***.php

А как во всём коде /models/***.php заменить на /models/***.model.php



Урок 5: Парсинг URL с помощью роутера. Класс Router.

На первом месте в URI может быть /admin/ или /default/. Если в URI есть /admin/ или /default/, то язык не переопределится.



Урок 6: Контроллеры.

Все контроллеры наследуются от класса /lib/controller.class.php. Конструктор этого класса получает массив $data.
